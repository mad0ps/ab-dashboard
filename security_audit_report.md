### КРИТ-4. Домен borelli-club.ru захвачен казино 1Win
**Риск:** КРИТИЧЕСКИЙ (репутационный + юридический)

Домен `borelli-club.ru`, ранее принадлежавший бренду, **полностью захвачен** нелегальным онлайн-казино **1Win**.

**Подтверждённые факты (глубокая проверка 22.03.2026):**

**Содержимое сайта:**

- Контент на русском языке, поддержка 20+ языков
- APK-файлы для скачивания мобильного приложения казино

**Техническая инфраструктура захвата:**
- Хостится за Cloudflare (защита от идентификации реального сервера)
- Работает на WordPress (`wp-json/` API активен в заголовках)
- HTTP автоматически редиректит на HTTPS (301)
- X-Frame-Options: SAMEORIGIN (защита от встраивания)
- Аффилиатные ссылки ведут на `youshalldopass.pro` и `xpst.online` (скачивание APK)
- Параметр аффилиата: `paramid=33329`, трафик-сайт: `bistrogorskoesp.ru`

**КРИТИЧЕСКАЯ НАХОДКА — индексация в Google:**
Поисковая выдача `site:borelli-club.ru` показывает **ОДНОВРЕМЕННО**:
- Старые страницы Borelli (одежда, пуховики, обувь) — Google помнит оригинальный контент

Это означает: **пользователи ищущие "Borelli Club купить детскую одежду" в Google видят результаты, ведущие на сайт казино**. Для детского бренда премиум-класса это юридическая и репутационная катастрофа.

**Факт об 1Win:** В 2024 году база 1Win была взломана — утекли данные 96.2 миллионов аккаунтов. Домен бренда Borelli ассоциируется с оператором, допустившим массовую утечку персональных данных.

**Действие:** Юридическое обращение к регистратору домена, жалоба в Роскомнадзор, письмо Cloudflare (abuse), запрос на деиндексацию через Google Search Console.

### КРИТ-7. PHPSESSID без флага Secure — alessandroborelli.ru
**Риск:** КРИТИЧЕСКИЙ

Куки сессии `PHPSESSID=...; path=/` отправляются **без флагов Secure и HttpOnly**:
```
set-cookie: PHPSESSID=3f7f491cdeb04e55baa4c7ae25ba57d7; path=/
```

При этом WooCommerce session cookie корректна (`secure; HttpOnly`), а вот PHP session — нет. Это означает, что ID сессии может быть перехвачен через незащищённое соединение или JavaScript-инъекцию.

**Стоимость:** 5 000 руб.

## 2. СЕРЬЁЗНЫЕ ПРОБЛЕМЫ (исправить в течение месяца)

### СЕР-1. Нет заголовков безопасности — ВСЕ 4 сайта
**Риск:** СЕРЬЁЗНЫЙ

Ни один из четырёх сайтов не отправляет критически важные HTTP-заголовки безопасности:

| Заголовок | .ru | BFG | IMP | .com |
|-----------|:---:|:---:|:---:|:----:|
| Strict-Transport-Security (HSTS) | -- | -- | OK | -- |
| X-Frame-Options | -- | -- | -- | -- |
| Content-Security-Policy | -- | -- | -- | -- |
| X-XSS-Protection | -- | -- | -- | -- |
| X-Content-Type-Options | -- | -- | -- | -- |

Только imperiya-detstva.ru имеет HSTS (`max-age=31536000`).

**Последствия:**
- Без HSTS: возможен даунгрейд HTTPS до HTTP (MITM-атака)
- Без X-Frame-Options: возможен clickjacking (встраивание сайта в iframe)
- Без CSP: нет защиты от XSS-инъекций
- Без X-Content-Type-Options: MIME-sniffing атаки

**Стоимость:** 10 000 - 30 000 руб. (настройка nginx для всех сайтов)

## 3. РЕКОМЕНДАЦИИ (желательно исправить)

### РЕК-1. Единая точка отказа — Битрикс-сайты
Оба Битрикс-сайта (borellifashiongroup.ru и imperiya-detstva.ru) на одном IP (79.133.182.182), одинаковый хеш CMS. Падение сервера = падение двух сайтов. Рекомендация: мониторинг uptime, план аварийного переключения.

### РЕК-2. Разные хостинг-провайдеры
- WordPress-сайты: Beget (ns1/ns2.beget.ru)
- Битрикс-сайты: AdminVPS (ns1.adminvps.ru)

Нет единого управления инфраструктурой. Рекомендация: консолидация или хотя бы единый мониторинг.

### РЕК-3. xmlrpc.php — WordPress
На обоих WP-сайтах xmlrpc.php отвечает 405. Хорошо, что не 200, но лучше полностью заблокировать (используется для brute-force и DDoS-усиления).

### РЕК-4. wp-cron.php публично доступен
